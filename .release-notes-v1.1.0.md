## What's new

- **Daily report timezone:** GitHub Actions now runs at **7:00 PM IST (GMT+05:30)**, Monday–Friday, using `timezone: Asia/Kolkata` (no manual UTC conversion).
- **Staying up to date:** README explains how to sync from upstream and watch for new releases.
- **Changelog:** See [CHANGELOG.md](https://github.com/vivek-kubvt/CUDashboard/blob/main/CHANGELOG.md) for version history.

## Upgrading (if you already cloned or forked)

```bash
git remote add upstream https://github.com/vivek-kubvt/CUDashboard.git   # once, if missing
git fetch upstream
git merge upstream/main
git push origin main   # only if you use your own fork on GitHub
```

Then:

1. If `package-lock.json` changed: `npm install --legacy-peer-deps`
2. If you use **GitHub Actions** for the daily report: confirm secrets `CURSOR_SESSION_TOKEN` and `GOOGLE_CHAT_WEBHOOK`, then run **Actions → Cursor Usage Report → Run workflow** once.

## Setup (new users)

See the [README](https://github.com/vivek-kubvt/CUDashboard#readme): session token, optional Google Chat webhook, `npm run dev` or scheduled workflow.

## Get notified of future updates

On this repo: **Watch → Custom → Releases**.
